解析Linux网络

發表於2024-07-10|更新於2024-07-10|Linux
|閱讀量:

好的，下面是详细的网络工具命令及其高级应用指南，包括每个命令的基本用法和高级用法。

1. tcpdump

tcpdump 是一个网络抓包工具，可以捕获并分析网络流量。

基本用法：

捕获接口 eth0 上的所有数据包。

高级用法：

捕获与特定主机 192.168.1.1 相关的数据包。

捕获接口 eth0 上所有通过端口 80 的数据包。

将捕获的数据包保存到文件 capture.pcap。

读取并分析 capture.pcap 文件中的数据包。

捕获所有带有 SYN 或 ACK 标志的 TCP 包。

2. lsof

lsof 显示系统中打开的文件，常用于查看打开的网络连接。

基本用法：

显示所有打开的网络连接。

高级用法：

显示所有使用端口 80 的网络连接。

显示特定进程 <PID> 打开的文件。

显示特定用户打开的文件。

显示哪个进程打开了指定文件。

3. net-tools

net-tools 包含 ifconfig、netstat 等工具。

基本用法：

显示接口 eth0 的配置信息。

显示所有活动的网络连接及其状态。

高级用法：

配置接口 eth0 的 IP 地址和子网掩码。

显示路由表信息。

显示网络接口统计信息。

显示所有监听的 TCP 端口及其关联的进程。

4. iproute2

iproute2 是 net-tools 的替代品，提供 ip 命令。

基本用法：

显示所有网络接口。

显示所有接口的 IP 地址信息。

高级用法：

为接口 eth0 添加 IP 地址。

启用接口 eth0。

设置默认网关。

禁用接口 eth0。

添加路由规则，使来自 192.168.1.0/24 的流量使用路由表 1。

5. NetworkManager

NetworkManager 是管理网络配置的工具。

基本用法：

交互命令窗口

查看当前设备的连接状态。

列出所有已保存的网络连接。

高级用法：

列出可用的 Wi-Fi 网络。

连接到指定的 Wi-Fi 网络。

添加一个名为 my-eth0 的以太网连接。

修改连接 my-eth0 的 IP 地址和网关。

启用连接 my-eth0。

6. firewalld

firewalld 提供动态的防火墙管理工具。

基本用法：

启动 firewalld 服务。

查看当前活动的区域。

列出当前区域的所有规则。

高级用法：

永久打开端口 80。

重新加载防火墙配置。

将 http 服务添加到 public 区域。

永久关闭端口 80。

7. iptables

iptables 是 Linux 内核中的包过滤工具。

基本用法：

列出所有当前规则。

允许所有进入的 TCP 连接通过端口 80。

高级用法：

删除规则，阻止端口 80 的连接。

丢弃来自 192.168.1.1 的所有连接。

允许所有 ICMP 出站流量。

保存当前规则到文件。

从文件恢复规则。

8. nftables

nftables 是 iptables 的继任者。

基本用法：

列出当前规则集。

添加规则，允许通过端口 80 的 TCP 连接。

高级用法：

删除规则，阻止端口 80 的连接。

丢弃来自 192.168.1.1 的所有连接。

添加一个名为 mytable 的表。

在 mytable 表中添加一个名为 mychain 的链。

在 mychain 链中添加一个计数规则。

9. curl

curl 是用于传输数据的命令行工具。

基本用法：

下载文件 file。

获取 HTTP 响应头信息。

高级用法：

发送 POST 请求。

使用基本身份验证下载文件。

忽略 SSL 证书错误。

跟随重定向。

断点续传下载文件。

10. wget

wget 是另一个下载文件的命令行工具。

基本用法：

下载文件 file。

静默模式下载文件，不输出任何信息。

高级用法：

递归下载整个网站，并将文件保存到 ./local 目录中

。

断点续传下载文件。

限制下载速度为 100kB/s。

使用基本身份验证下载文件。

11. iptop

iptop 是一个实时显示网络流量的工具。

基本用法：

启动 iptop，显示实时网络流量。

高级用法：

显示特定接口 eth0 的网络流量。

显示来自特定子网 192.168.1.0/24 的流量。

显示发送到端口 80 的流量。

这些命令涵盖了大多数网络管理和诊断的日常需求。每个工具都有其高级用法和配置选项，可以根据具体需求进行调整和应用。
